Quality of Service

Quality of Service (QoS) is a set of technologies that enable reliable and predictable network performance by prioritizing network traffic.

Application Requirement

Application Requirement is a QoS technique that refers to the different needs and prioritize of different applications that run on the network.It consist the 4 parameter. They are Bandwidth,Delay,Jitter,Loss.

Traffic Shaping

Traffic shaping is a QoS technique that controls the rate of traffic sent or received by a network device. This can help to prevent network congestion and improve overall network performance.

Packet Scheduling

Packet scheduling is a QoS technique that determines how packets are sent or received by a network device. This can help to ensure that critical traffic is prioritized over less important traffic.

Admission Control

Admission control is a QoS technique that limits the amount of traffic that is allowed on a network. This can help to prevent network congestion and ensure that network performance remains reliable.

Integrated Service

Integrated services is a QoS model that provides end-to-end quality of service by reserving network resources for specific applications or services. This can be done using protocols such as RSVP.

Differentiated Service

Differentiated services is a QoS model that provides different levels of service to different types of traffic. This can be done using techniques such as classifier service and expedited service.
